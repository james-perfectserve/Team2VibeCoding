# Rock Paper Scissors Game — 1.5 Hour Hackathon

## Core Requirements
- Multiplayer, real-time
- Player joins → enters lobby → matched with opponent → game begins

## Stretch Goals
- **WebGL** — 3D hands, particle effects, shaders
- **Leaderboards** — persistent stats, rankings, streaks
- **Terminal UI** — ncurses, ASCII art, retro vibes
- **SSH game server** — `ssh play@yourserver.com`
- **AI opponent** — pattern recognition, adaptive difficulty
- **Tournament mode** — brackets, spectators, replays

> If you finish in 10 minutes... you are not done. Make it amazing.
